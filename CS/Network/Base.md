## 네트워크에 대한 기본적인 지식에 대한 것을 정리하는 공간


* URL과 URI의 차이점
<br>

## URL이란?
<br>

### URL(Uniform Resource Locator)
* 웹주소, 컴퓨터 네트워크상에서 리소스가 어디에 위치하는지 알려주기 위한 규약
* 식별자 + 위치
* https://elancer.co.kr -> 프로토콜 + URI 형태
<br>

## URI이란?
<br>

### URI(Uniform Resource Identifier)
* 네트워크 통합자원의 위치를 나타내기 위한 규약
* 식별자
* URL 보다 더 포괄적 개념
* ex) elancer.co.kr
